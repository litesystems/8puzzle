Installation
------------
1. `git clone https://github.com/litesystems/8puzzle.git`
2. `cd 8puzzle`
3. `pip install .`

Example
-------
### Input
```
eightpuzzle -o out.pdf astar manhattan "[[4,1,2],[6,0,3],[7,5,8]]"
```
